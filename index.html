Conceito:

HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página web. É o componente básico da web e permite a criação de websites.

Principais conceitos do HTML

HTML é uma série de elementos que marcam diferentes partes do conteúdo.

O HTML define a disposição de elementos como textos, imagens, links e botões

O HTML permite a construção de páginas web simples ou complicadas.

O HTML permite a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

O HTML é a base para outros recursos interativos, como cliques em botões ou links em textos.

Client x server

Na parte de client x server, pode-se observar que um leva o outro. Não há possibilidades de criar um site e o client não poder dar seu feedback. Nessa parte do curso, entende-se o suficinte que não pode-se trabalhar para si próprio, sempre tem que pensar nos clientes e fazer do seu site, o mais querido por eles.
